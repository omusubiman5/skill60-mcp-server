## 🎯 v3.1の設計思想

### データとLLMの分離
```
┌─────────────────┐
│   Claude API    │ ← LLM分析・アドバイス生成
│  (LLM処理層)    │    - ヨシコの声
│                 │    - 方言変換
│                 │    - 健康アドバイス
└────────┬────────┘    - スキル評価
         │
         ↓
┌─────────────────┐
│  SKILL60+ MCP   │ ← データ取得専用
│   Server v3.1   │    - ニュース取得
│                 │    - 年金情報取得
│                 │    - 求人検索
└────────┬────────┘    - 天気予報取得
         │
         ↓
┌─────────────────┐         ┌─────────────────┐
│  外部サイト/API  │         │  MongoDB Atlas  │
│  (データソース)  │         │  (エラーログ)   │
└─────────────────┘         └─────────────────┘
```

### 全ツールの共通仕様
- **Description**: "このツールは生データを返すのみ。LLM側で[X]を生成してください。"
- **エラーハンドリング**: MongoDB logError統合
- **フェールセーフ**: MongoDB未接続でもツール動作
